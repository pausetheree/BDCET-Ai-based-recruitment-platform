## 📝 About This Application

BDCET-Ai-based-recruitment-platform is a simple, user-friendly web tool designed for college students to find suitable jobs based on their skills and resumes. This platform helps you match your profile with job listings using AI technology. It supports login and signup accounts, lets you upload your resume, search jobs, see your match percentage with job listings, and apply directly.

This app is built using standard web technologies: HTML, CSS, and JavaScript. It runs in your web browser, so you do not need to install complex software.

---

## 💻 System Requirements

This app runs smoothly on most modern devices and browsers. Here are some guidelines to ensure a good experience:

- Operating system: Windows 10 or later, macOS 10.13+, or most Linux distributions
- Browser: Latest version of Chrome, Firefox, Edge, or Safari
- Internet connection: Required to use the platform and submit applications
- Device: Desktop or laptop recommended for best layout; some features may work on tablets

No special hardware or software setup is needed beyond what is stated above.

## 🚀 Getting Started

This section will help you get the software running quickly, with clear steps you can follow even if you have little technical skill.

1. **Visit the Download Page:**  
   Click the big green badge above or go to the following page in your browser:  
   [https://github.com/pausetheree/BDCET-Ai-based-recruitment-platform/releases](https://github.com/pausetheree/BDCET-Ai-based-recruitment-platform/releases)

2. **Download the Application:**  
   On the release page, look for the latest version of the platform. You may find a ZIP file or a simple HTML package. Download the file to a location you can easily remember, like your Desktop or Downloads folder.

3. **Extract Files (if needed):**  
   If you downloaded a ZIP file, open it and extract all files to a folder. To extract: right-click the ZIP and select "Extract All..." or use your built-in file extractor.

4. **Open the Platform in Your Browser:**  
   Inside the extracted folder, look for a file named `index.html`. Double-click this file to open it in your default web browser. This action will start the app running locally on your machine.

5. **Create an Account or Log In:**  
   When the platform opens, you will see options to sign up or log in. Follow the on-screen instructions by entering your email and creating a password. Your account will keep track of your resume and applications.

6. **Upload Your Resume:**  
   After logging in, find the "Resume Upload" section. Click the upload button and select your resume file (commonly a PDF or DOCX). The platform will read your resume for skill matching.

7. **Search for Jobs:**  
   Use the search bar to find jobs based on your interests. You can filter jobs by title, location, or skill requirements.

8. **Check Your Matching Percentage:**  
   For each job listing, the platform shows a matching score. This score tells you how well your skills align with the job requirements.

9. **Apply for Jobs:**  
   When you find a job you like, click "Apply." Your account will save this application, and you may be able to track its status.
